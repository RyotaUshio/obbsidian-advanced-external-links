# Obsidian Advanced External Links

This [Obsidian.md](https://obsidian.md) plugin lets you paste URLs cleverly.

- Automatically fetch the title of the target webpage
- ***Format a link to a text highlight with highly customizable templates***: you can annotate webpages without read-it-later apps like Omnivore.

## Installation

Since this plugin is still in its beta, it's not available in the community plugin browser yet.

But you can install the latest release using [BRAT](https://github.com/TfTHacker/obsidian42-brat).

1. Install the latest version of BRAT and enable it.
2. _(Optional but highly recommended)_ In the BRAT settings, turn on `Auto-update plugins at startup` at the top of the page.
3. Open the following URL in the browser: `obsidian://brat?plugin=RyotaUshio/obsidian-advanced-external-links`.
4. Click the "Add Plugin" button.
